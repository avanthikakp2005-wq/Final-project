                               Social Media Usage and Academic Performance Analysis

Overview
    End-to-end exploratory data analysis (EDA) of student data using Python to understand how social media usage affects academic            performance, study habits, and sleep patterns.

Key Objectives
	•	Analyze the relationship between social media usage and GPA
	•	Identify behavioral patterns affecting academic performance
	•	Evaluate the impact of study hours and sleep on GPA
	•	Segment students based on usage and performance
	•	Derive actionable academic insights

Dataset
~Student survey dataset containing academic and behavioral variables such as GPA, social media usage (hours/day), study hours, sleep duration, and usage purpose (educational vs entertainment).

Analysis Highlights
	•	High social media usage (>4 hours/day) is linked to lower GPA
	•	Study hours show a strong positive impact on academic performance
	•	Social media usage negatively affects study time (time displacement effect)
	•	Educational usage has a neutral/slightly positive effect
	•	Entertainment usage shows a stronger negative impact
	•	Night-time usage reduces sleep, indirectly lowering GPA
	•	Moderate usage does not significantly harm performance

Statistical Insights
	•	Social media usage shows a moderate negative correlation with GPA
	•	Study hours and sleep duration are strong positive predictors
	•	Regression analysis confirms social media as a significant negative factor
	•	Model explains a meaningful portion of GPA variation

Customer (Student) Segmentation
	•	High Performers: Low usage, high study time, high GPA
	•	Balanced Users: Moderate usage, stable performance
	•	Heavy Users: High usage, low study time, lower GPA

Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

Outcome
The analysis shows that behavioral factors such as study time, sleep, and social media usage patterns play a more critical role than general habits. It highlights the importance of balanced and purposeful social media use for maintaining strong academic performance.
